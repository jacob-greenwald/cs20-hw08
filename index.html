The API used is the RANDOM USER GENERATOR. It generates random user data.

The API can be found on randomuser.me

The only options used by the API request were to return the data in the default JSON format.

This API could be useful when generating a profile on a suspicious website where you might not want to give your real information.
It may also be used to generate characters for a story.
